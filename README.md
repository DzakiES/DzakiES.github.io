## 🚀 Deployment

### GitHub Pages Setup:
1. Push all files to the `main` branch
2. Go to repository **Settings** → **Pages**
3. Source: Deploy from **main** branch
4. Root directory: **/ (root)**
5. Click **Save**
6. Wait 1-2 minutes for deployment
7. Access your site at `https://dzakies.github.io`

### Local Development:
```bash
# Clone the repository
git clone https://github.com/DzakiES/DzakiES.github.io.git

# Navigate to the directory
cd DzakiES.github.io

# Open index.html in your browser
# Or use a local server:
python -m http.server 8000
# Then visit http://localhost:8000
```

## 🎨 Customization

### Update Personal Information:
Edit `index.html`:
- Line 42-48: Hero section text
- Line 85-95: About section
- Line 186-188: Contact information

### Change Colors:
Edit `css/style.css` (lines 5-16):
```css
:root {
    --primary-color: #2563eb;    /* Main color */
    --secondary-color: #8b5cf6;  /* Secondary color */
    /* ... other colors */
}
```

### Add New Projects:
Copy a project card in `index.html` (lines 170-195) and modify:
- Project title
- Description
- Tech stack tags
- GitHub link
